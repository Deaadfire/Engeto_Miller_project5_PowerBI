# Engeto_Miller_project5_PowerBI #

Projekt znázorňuje aktuální (data k 24.1.2026) přehled všech státních mateřských a základních škol v Brně. Zaměřuje se především na jejich rozpoložení v jednotlivých městských částí Brna a mapuje jejich spádové oblasti.  
Spádové oblasti jako objekty zde představují jednotlivé adresy - jeden objekt tedy může být velký panelový dům s velkým počtem obyvatel či malý rodinný dům.  
Projekt pracuje pouze s objekty spádové oblasti a nikoli s počtem obyvatel - spíše mapuje, než aby porovnával kapacitu.
